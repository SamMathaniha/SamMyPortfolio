Sam Store - E-Commerece Website




"Sam Store" is a full-featured e-commerce website built using Laravel 11, designed to provide a seamless shopping experience. The project includes a robust client-side interface with secure login, shopping cart, and both payment-on-delivery and online payment options via Stripe. The platform also features email verification for enhanced security and a powerful admin dashboard for managing products, orders, and users. This project showcases advanced web development skills, including database management, multi-authentication, and integration of modern tools like Breeze, Toastr, and SweetAlert.
